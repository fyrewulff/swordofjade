Browse to pick a shop.

You can type the name of a shop or type the shop's ID number to make the cursor jump to it.

Click the "+New" button or press CTRL+N to add a new shop.

Right-click for a pop-up menu that lets you choose if you want to "Pick" this shop, or if you want to edit it.

Or press CTRL+ENTER or SHIFT+ENTER or Ctrl+E to edit.

Press CTRL+F or click "Filter" if you want to filter the list to only show shops that match a certain string of text.
